# To-Do List Application

A simple and interactive command-line to-do list application that allows users to add, delete, and complete tasks. This project aims to help users manage their tasks in a straightforward and efficient way using the terminal.

## Getting Started

These instructions will help you get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need to have Python installed on your machine. If you don't have it already, follow these steps to install it:

- [Install Python](https://www.python.org/downloads/) - Follow the installation instructions for your operating system.
- Install `setuptools`:
  ```bash
  pip install setuptools
